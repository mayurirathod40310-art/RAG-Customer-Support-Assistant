# RAG-Based Customer Support Assistant

This project implements a Retrieval-Augmented Generation (RAG) based customer support assistant using LangChain, ChromaDB, and LangGraph.

## Features
- PDF document processing
- Text chunking
- Embedding generation
- ChromaDB vector storage
- Retrieval-Augmented Generation (RAG)
- LangGraph workflow
- Human-in-the-Loop (HITL) escalation

## Technologies Used
- Python
- LangChain
- ChromaDB
- HuggingFace
- LangGraph
